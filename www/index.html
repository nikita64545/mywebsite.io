О проекте

Этот сайт создан в рамках обучения веб-разработке. Я хочу продемонстрировать свои навыки в создании простого, но функционального вебсайта.

Атрибут img src="" alt="" позволяет добавлять изображения.

Для начала,я использовал только HTML. В дальнейшем постараюсь добавить CSS для улучшения внешнего вида и интерактивности.

Особенности этого примера

  • Простая структура страницы
  • Использование основных HTML-тегов
  • Комментарии для объяснения кода
  • Места для будущих улучшений (CSS, JS)

Например, изображение выше - это переход на сайт МАОУ СОШ №217 Атрибут target="_blank" открывает ссылку в новой вкладке.
